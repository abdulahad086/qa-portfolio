# QA Automation Portfolio - Demo Web Shop

Welcome to my comprehensive QA Engineering Portfolio!

This repository was created to demonstrate a complete "End-to-End QA Workflow" on the popular Demo Web Shop e-commerce website [https://demowebshop.tricentis.com/](https://demowebshop.tricentis.com/). It encompasses both manual software testing artifacts and automated scripts.

## 🛠 Tech Stack & Tools Used

- **Testing Framework:** Cypress (JavaScript)
- **Version Control:** Git & GitHub
- **Development Environment:** Visual Studio Code
- **Testing Strategy:** E2E Automation, Functional Manual Testing

## 📁 Repository Structure

- **/test-cases**: Contains manual test case documentation mapping out the application's core functionality.
- **/bug-reports**: Contains sample UI & Functional bug reports with designated Severity/Priority and Steps to Reproduce.
- **/automation**: A complete Cypress automation framework (E2E testing).
  - `cypress/e2e/login.cy.js`
  - `cypress/e2e/register.cy.js`
  - `cypress/e2e/search.cy.js`
  - `cypress/e2e/cart.cy.js`
  - `cypress/e2e/checkout.cy.js`

## 🚀 How to Run the Automation Tests

### 1. Prerequisites

Ensure you have Node.js installed on your machine.

### 2. Setup

Navigate into the \`automation\` folder and install dependencies:
\`\`\`bash
cd automation
npm install
\`\`\`

### 3. Running tests

**To run the tests with the Cypress UI (Interactive Mode):**
\`\`\`bash
npx cypress open
\`\`\`

**To run all tests headlessly in the terminal:**
\`\`\`bash
npx cypress run
\`\`\`
