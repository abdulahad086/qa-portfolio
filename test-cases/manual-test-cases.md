# Manual Test Cases - Demo Web Shop

This document outlines the manual test scenarios designed for the Demo Web Shop application to ensure core functionalities work as expected.

| Test ID   | Module   | Title                     | Pre-conditions                               | Steps to Reproduce                                                                                                                       | Expected Result                                                                                        | Status |
| :-------- | :------- | :------------------------ | :------------------------------------------- | :--------------------------------------------------------------------------------------------------------------------------------------- | :----------------------------------------------------------------------------------------------------- | :----- |
| **TC_01** | Account  | Successful Valid Login    | User must be registered.                     | 1. Open homepage.<br>2. Click 'Log in'.<br>3. Enter valid Email & Password.<br>4. Click 'Log in' button.                                 | User is successfully authenticated, and 'Log out' link is visible in the header.                       | Pass   |
| **TC_02** | Account  | Invalid Login Attempt     | User must be on the Login page.              | 1. Enter random/incorrect email.<br>2. Enter random password.<br>3. Click 'Log in' button.                                               | Validation error message displayed: "Login was unsuccessful. Please correct the errors and try again." | Pass   |
| **TC_03** | Product  | Search Functionality      | At least one item in shop matches the query. | 1. Type 'computer' in the search bar.<br>2. Click 'Search' button.                                                                       | Search results page loads displaying at least 1 related product.                                       | Pass   |
| **TC_04** | Cart     | Add item to Shopping Cart | User is on a product listing page.           | 1. Click 'Add to cart' on a product.<br>2. Wait for confirmation message.<br>3. Open Shopping Cart.                                      | The cart page reflects the added item with correct quantity and price.                                 | Pass   |
| **TC_05** | Checkout | Guest Checkout Completion | Cart has at least 1 item in it.              | 1. Go to Cart and Click Checkout.<br>2. Select 'Checkout as Guest'.<br>3. Fill standard shipping & payment forms.<br>4. Click 'Confirm'. | Order success page appears and a valid Order ID is generated.                                          | Pass   |
